# Client Side Rendering React + TypeScript + StyledComponents with Webpack

This is my boilerplate so I don't have to deal wtih webpack setup again, for a while.

Includes:

- React
- TypeScript
- React Router
- Lodash
- Dotenv
- Express
- Jest
- ESLint with TypeScript plugins
- Rimraf
- Webpack with:
  - ts-loader
  - file-loader
